# SQL HR Analytics Case Study

## About the Project
Analyzed IBM HR Attrition dataset with 1,470 records 
using MySQL to answer 15 business questions covering 
attrition trends, salary analysis, and workforce demographics.

## Tools Used
MySQL | MySQL Workbench

## Dataset
IBM HR Analytics Attrition Dataset — 1,470 rows, 35 columns
Source: Kaggle

## Business Questions Covered
1. Total number of employees
2. Attrition count and rate
3. Department-wise employee count
... (list all 15)

## Key Insights
- 16% overall attrition rate
- Sales department has highest attrition at 20%
- Employees doing overtime are 3x more likely to leave
- Employees who left earned 20% less than those who stayed
